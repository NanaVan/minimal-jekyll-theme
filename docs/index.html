Introducing Minimal

Minimal Jekyll Theme is an open source Jekyll theme by Desired Persona. Lets take a look at what you can achieve starting with the most minimal of layouts.

  • A minimal post with a title
  • A post with a title and subtitle
  • A post with post meta above title
  • A post with post meta and an author bio
  • A post with share icons.
  • A post with categories and tags
  • A post with comments turned off
  • This post has been updated
  • Change post background color
  • Post formatting
  • Post with minimal hero header
  • Post with hero image
  • Post with hero image and call-to-action

You can find out more info about customising the Minimal theme in the theme documentation, as well as basic Jekyll usage documentation at jekyllrb.com.
